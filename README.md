# Animal Identification Game

## Objective
The objective of this project is to identify animals based on the image displayed on the screen.

## Features
- Displays animal images
- Multiple choice answers
- Score tracking
- Interactive GUI

## Technologies Used
- Python
- Tkinter
- Pillow (PIL)

## Installation

Install Pillow:

pip install pillow

## Run Project

python animal_game.py

## Project Concepts
- Images
- Buttons
- Event Handling
- Scoring System
- GUI Programming
